Html setup

When makeing an HTML file you must start with first telling the document what language you'll be writing in by using. <!DOCTYPE html> This will tell the page what language to read.

Next is the <html></html> element also refered to as the root element. this will wrap your doument in html

Next we have the <head> element. this will wrap everything you want that isn't content on the page. your viewer won't see this section. this can be things like search terms or css style sheets.

Last we have the <body> element. this is where you will put the contents of your site. everything from navigation bars to images and paragrphs will go here.
